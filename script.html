script

Permite embeber código ejecutable en el documento HTML. Estos programas son interpretados y ejecutados por el navegador.
No tiene valor semántico -type indica el lenguaje. -src indica la url del programa. 

        
            <script>
                function sumar (a, b) {
                var resultado = a + b;
                return resultado;
                }
            </script>
